## Run Locally

Clone the project

```bash
  git clone https://github.com/sudeepkudari0/Dsa-Web.git
```

Go to the project directory

```bash
  cd Dsa-Web
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
